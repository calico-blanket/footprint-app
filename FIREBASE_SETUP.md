# Firebase設定ガイド

このアプリ（Footprint App）は、データを保存するためにGoogleの「Firebase（ファイアベース）」というサービスを使用します。
以下の手順で設定を行い、必要な情報を取得してください。

## 手順 1: Firebaseプロジェクトの作成
1.  [Firebaseコンソール](https://console.firebase.google.com/) にアクセスします。
2.  Googleアカウントでログインします。
3.  **「プロジェクトを作成」**（または「プロジェクトを追加」）をクリックします。
4.  プロジェクト名を入力します（例: `footprint-app`）。
5.  「続行」をクリックします。
6.  Googleアナリティクスは「無効」にして「プロジェクトを作成」をクリックします（有効でも構いませんが、設定が少し増えます）。
7.  作成が完了したら「続行」をクリックします。

## 手順 2: アプリの登録と設定情報の取得
1.  プロジェクトの概要ページ（トップページ）の中央にあるアイコンから **「</> (ウェブ)」** をクリックします。
2.  アプリのニックネームを入力します（例: `Web App`）。
3.  「Firebase Hosting も設定します」のチェックは **外したまま**、「アプリを登録」をクリックします。
4.  **「Firebase SDK の追加」** という画面が表示されます。
5.  `const firebaseConfig = { ... };` と書かれた部分を探します。以下のような情報が表示されています。

    ```javascript
    apiKey: "AIzaSy...",
    authDomain: "footprint-app-....firebaseapp.com",
    projectId: "footprint-app-...",
    storageBucket: "footprint-app-....firebasestorage.app",
    messagingSenderId: "123456...",
    appId: "1:123456..."
    ```

6.  この情報をメモ帳などにコピーしておいてください。後で使用します。
7.  「コンソールに進む」をクリックします。

## 手順 3: Authentication（認証）の設定
1.  左側のメニューから **「構築」** > **「Authentication」** をクリックします。
2.  **「始める」** をクリックします。
3.  「ログインプロバイダ」タブで **「Google」** をクリックします。
4.  右上のスイッチを **「有効」** にします。
5.  「プロジェクトのサポートメール」で自分のメールアドレスを選択します。
6.  **「保存」** をクリックします。

## 手順 4: Firestore（データベース）の設定
1.  左側のメニューから **「構築」** > **「Firestore Database」** をクリックします。
2.  **「データベースの作成」** をクリックします。
3.  場所（ロケーション）を選択します（日本なら `asia-northeast1` (Tokyo) がおすすめですが、デフォルトのままでも動きます）。
4.  「次へ」をクリックします。
5.  **「本番環境モードで開始する」** を選択し、「作成」をクリックします。

## 手順 5: Storage（画像保存）の設定
1.  左側のメニューから **「構築」** > **「Storage」** をクリックします。
2.  **「始める」** をクリックします。
3.  「本番環境モードで開始する」を選択し、「次へ」をクリックします。
4.  ロケーションはそのままで「完了」をクリックします。

---

## 最後に: 設定情報を伝える
手順2でコピーした情報を、チャットで教えてください。
また、アプリにログインできる人を制限するため、**「ログインに使用したいGoogleメールアドレス」** も教えてください。

例：
```
apiKey: AIzaSy...
authDomain: ...
...
メールアドレス: myname@example.com
```
